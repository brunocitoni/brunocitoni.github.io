How to:
With headphones on, select the receiver position you would like to audition then check the Real-Time checkbox

Alternatively select a sound source and a receiver position, then click listen. To change receiver position, make sure you stop and press listen again.

Move the slider to appreciate the difference between dry signal and wet signal while either real-time input or a source is playing.
